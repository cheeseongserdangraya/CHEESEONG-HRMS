旧数据搬迁工具

只用一次:把旧版 FIRSTONE_HRMS.html(localStorage 版)导出的 JSON 备份,搬到这个 Supabase 数据库。只有 admin 账号能用。搬完之后这个页面可以留着不管,不影响系统。

1. 选择备份文件

在旧版 HTML 工具里点右上角「导出备份 (JSON)」下载的那个文件,例如 firstone_hrms_backup_2026-07-14.json

2. 确认无误后开始搬迁

⚠️ 这个操作只应该做一次。重复执行会把同一批员工/记录重复插入一遍(变成两份)。如果不确定,先去 Supabase Table Editor 检查 employees 表是不是空的。
